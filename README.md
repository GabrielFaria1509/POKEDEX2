# 🗺️ Pokédex App

Uma aplicação web interativa e responsiva que replica o funcionamento de uma Pokédex clássica. O projeto consome dados dinâmicos diretamente da **PokeAPI** para listar, buscar e exibir informações dos Pokémon em tempo real.

Este projeto foi desenvolvido como parte dos meus estudos práticos de front-end para consolidação de fundamentos de desenvolvimento web e manipulação de APIs.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica da interface e dos elementos da Pokédex.
* **CSS3:** Estilização customizada, posicionamento, uso de variáveis para cores dos tipos e responsividade para diferentes tamanhos de tela.
* **JavaScript (ES6+):** Lógica de programação, consumo assíncrono da API (Fetch API), tratamento de dados e manipulação dinâmica do DOM.

---

## ⚙️ Principais Funcionalidades

* **Busca Dinâmica:** Encontre qualquer Pokémon pesquisando diretamente pelo nome ou pelo número de identificação (ID).
* **Navegação Sequencial:** Botões de "Anterior" (*Prev*) e "Próximo" (*Next*) para navegar linearmente pela lista de Pokémon.
* **Consumo de API Assíncrono:** Interface atualizada instantaneamente sem a necessidade de recarregar a página, buscando imagens e dados oficiais direto da PokeAPI.
* **Interface Temática:** Design visual limpo e nostálgico baseado no dispositivo original do anime.
